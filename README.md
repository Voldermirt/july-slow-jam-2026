# July-slow-jam-2026
## Description
A silly 2D delivery platformer where you dodge bombs to deliver items.
## Controls
**a** - Left\
**d** - right\
**s** - down\
**space** - jump\
**e** - interact
## Features
Deliver Orders\
Dodge Bombs
## Built with
* [Godot Engine](https://godotengine.org)
* GDScript
